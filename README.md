# customer_shopping_behavior_analysis
Retail customer behavior analysis using Python, Pandas and MySQL.

# Project

This project analyzes retail customer shopping behavior using Python (Pandas) and MySQL.

The objective was to uncover business insights related to:

1.Customer purchasing behavior
2.Revenue contribution by demographics
3.Product performance
4.Discount effectiveness
5.Customer loyalty
6.Subscription behavior

# Tools Used:
1.Python
2.Pandas
3.Jupyter Notebook
4.MySQL Workbench
5.SQLAlchemy
6.PyMySQL


## Data Preparation

### Data Cleaning

- Imported customer shopping dataset using Pandas
- Handled missing values in review ratings using category-wise median imputation
- Standardized column names for SQL analysis
- Created age group segments for customer analysis

### Data Migration

- Connected Python to MySQL using SQLAlchemy
- Loaded cleaned data into MySQL for SQL-based analysis

## Business Questions Answered

### Revenue Analysis

- What is the total revenue generated by male vs female customers?

### Discount Analysis

- Which customers used discounts but still spent more than the average purchase amount?
- Which products have the highest percentage of purchases with discounts applied?

### Product Analysis

- What are the top 5 highest-rated products?
- What are the top 3 most purchased products within each category?

### Customer Analysis

- Do subscribed customers spend more than non-subscribers?
- Are repeat buyers more likely to subscribe?
- What is the revenue contribution of each age group?

### Logistics Analysis

- Compare average purchase amount between Standard and Express shipping.
